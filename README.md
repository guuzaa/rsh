## Usage

### Run

```bash
cargo run
```

### Build

```bash
cargo build --release
# run the binary
./target/release/rsh
```

### Builtins

- `cd` - Change the current directory.
- `help` - Display help information.
- `exit` - Exit the shell.
- Add your own builtins.
